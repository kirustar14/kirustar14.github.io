## 📸 How to Use

### Document Infrastructure (Before Drywall)

```
1. Tap (+) → Create Scene → Name it
2. Tap "New Capture" → Take Photo
3. Select Anchor Point (pick a corner/edge)
4. Add Note → Save
```

**✅ Good anchor points:** Corners, edges, fixtures  
**❌ Bad anchor points:** Center, unmarked spots

---

### View AR Overlay (After Drywall)

```
1. Open Scene → Tap "View AR"
2. Point camera at wall
3. Take Snapshot
4. Select Same Anchor Point
5. See overlay of hidden MEP!
```

---

## 💡 Pro Tips

**When to Capture:**
- ✓ After rough-in
- ✓ Before drywall
- ✓ After any MEP work
- ✓ Multiple angles of same area

**Anchor Point Selection:**
- ✓ Choose something permanent
- ✓ Pick corners or edges
- ✓ Use same point every time
- ✓ Something visible after drywall

**Notes to Add:**
- Type of MEP (water, electric, gas)
- Size/rating
- Who installed it
- Date

## 💬 Common Questions

**Q: How many photos can I take?**  
A: Unlimited! Take as many as you need.

**Q: Will this work offline?**  
A: You need internet to save, but you can take photos offline and they'll upload later.
