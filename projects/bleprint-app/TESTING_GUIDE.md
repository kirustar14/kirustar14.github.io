# BLEprint Field Testing Guide

## For Construction Workers Testing the App

### What You're Testing

BLEprint helps you document MEP systems (pipes, wires, ducts) before they get hidden behind walls, then shows you where they are when you need to drill later.

**Goal:** Save time, prevent mistakes, avoid hitting critical infrastructure.

---

## Test Scenarios

### Scenario 1: First Time Use (5 minutes)

**What to test:**
1. Download and install the app
2. Create an account
3. Sign in
4. Create your first scene

**What to check:**
- [ ] App opens without crashing
- [ ] Sign up process is clear
- [ ] You can create a scene easily
- [ ] Scene name makes sense

**Feedback questions:**
- Was anything confusing?
- How long did it take?
- Would you do this on a busy construction site?

---

### Scenario 2: Document a Wall (10 minutes)

**What to test:**
1. Go to an unfinished wall with visible MEP
2. Open your scene
3. Take a photo of the exposed infrastructure
4. Select a corner or edge as anchor point
5. Add notes about what's visible
6. Save the capture

**What to check:**
- [ ] Camera works properly
- [ ] Photo quality is good enough
- [ ] Anchor point selection makes sense
- [ ] You can add helpful notes
- [ ] Capture saves successfully

**Take photos of:**
- Exposed electrical conduit
- Visible water pipes
- HVAC ducts
- Any other MEP systems

**Feedback questions:**
- Is the anchor point selection clear?
- Would you remember to do this during construction?
- What features are missing?

---

### Scenario 3: AR Overlay (15 minutes)

**What to test:**
1. Take 2-3 photos of the same wall over time
2. After wall is finished (or simulate with hand covering camera)
3. Use "View AR" feature
4. Take a snapshot
5. Select the same anchor point
6. View the overlay

**What to check:**
- [ ] AR overlay appears
- [ ] Alignment seems accurate
- [ ] You can see hidden infrastructure
- [ ] It's useful for avoiding mistakes

**Feedback questions:**
- Is the overlay helpful?
- Can you see enough detail?
- Would this prevent you from hitting a pipe?
- What would make it better?

---

### Scenario 4: Real-World Use Case (30 minutes)

**What to test:**
Pick a real task you're doing:
- Installing anchors
- Running new conduit
- Drilling for mounting
- Any task where you need to avoid MEP

**Process:**
1. Before: Document current MEP with photos
2. After drywall: Use AR to see where MEP is
3. Complete your task safely
4. Report if it helped

**What to check:**
- [ ] Prevented a utility strike?
- [ ] Saved time finding infrastructure?
- [ ] More confident in drilling location?
- [ ] Would use again?

## Testing Tips

### For Best Results

1. **Test in real conditions**
   - Actual construction site
   - With gloves on
   - In different lighting
   - While actually working

2. **Try to break it**
   - Take 100 photos
   - Switch between scenes rapidly
   - Use offline/online
   - Kill and restart app

3. **Think long-term**
   - Would you use this every day?
   - Would you train new workers?
   - Would you pay for this?

### Common Scenarios to Test

- [ ] Morning start - document new wall
- [ ] Lunch break - check previous captures
- [ ] Afternoon - use AR before drilling
- [ ] End of day - review all scenes
- [ ] Next day - verify data persists
