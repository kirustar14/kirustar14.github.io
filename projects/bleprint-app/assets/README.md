# Assets Directory

This directory contains app assets like icons and splash screens.

## Required Assets

### Icon (icon.png)
- Size: 1024x1024 px
- Format: PNG
- Background: Your brand color or white
- Content: BLEprint logo

### Splash Screen (splash.png)
- Size: 1284x2778 px (portrait)
- Format: PNG
- Background: Your brand color
- Content: BLEprint logo centered

### Adaptive Icon (adaptive-icon.png)
- Size: 1024x1024 px
- Format: PNG
- For Android adaptive icons

### Favicon (favicon.png)
- Size: 48x48 px
- Format: PNG
- For web version

## Creating Assets

### Quick Method (Placeholder)
You can use solid color placeholders for development:

1. **Icon**: Create a 1024x1024 blue square with "BP" text
2. **Splash**: Create a 1284x2778 blue screen with "BLEprint" text
3. **Favicon**: Resize icon to 48x48

### Professional Method
1. Design your logo in a vector editor (Figma, Illustrator, etc.)
2. Export at the required sizes
3. Ensure proper padding and safe areas
4. Test on both light and dark backgrounds

## Using Your Own Assets

Simply replace the files in this directory with your own, keeping the same filenames:
- `icon.png`
- `splash.png`
- `adaptive-icon.png`
- `favicon.png`

## Generating from Source

If you have a logo SVG, you can use online tools:
- [MakeAppIcon](https://makeappicon.com/)
